Object-Oriented To-Do List
This project is an object-oriented version of my To-Do List application. It was developed using JavaScript classes to manage tasks in a more organized and reusable way. The program allows users to add, complete, delete, and display tasks while also validating user input.

Features
- Add a task
- Mark a task as completed
- Delete a task
- Display all tasks
- Input validation for empty task names

Files Included
- todo.js
- Documentation.pdf

Technologies Used
- JavaScript
- Node.js
- Visual Studio Code

How to Run
1. Install Node.js.
2. Download or clone the repository.
3. Open the project folder in Visual Studio Code.
4. Open Terminal and run:
```bash
node todo.js
```
5. The program output will be displayed in the terminal.

Reflection
Using classes made the code more organized by grouping related data and methods together. The separation between the Task and TodoList classes improved readability and made the program easier to maintain and extend in the future.

What I Learned
Through this project, I learned how to use classes, objects, constructors, and methods in JavaScript. I also learned the importance of input validation and how object-oriented programming can improve code organization and reusability.
